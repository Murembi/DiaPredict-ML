# Diabetes Prediction System

This project is a machine learning model that predicts the likelihood of diabetes based on medical input features such as glucose level, BMI, age, and blood pressure.

## Project Goal

To build a predictive system that helps identify individuals at risk of diabetes early using data-driven techniques.

## Dataset

The model is trained on a structured medical dataset containing patient health indicators relevant to diabetes diagnosis.

## Technologies Used

* Python
* Pandas & NumPy
* Scikit-learn
* Jupyter Notebook

## Model

A classification model is used to predict whether a patient is likely to have diabetes or not.

## How It Works

1. Load and clean the dataset
2. Perform exploratory data analysis
3. Train a machine learning model
4. Evaluate performance
5. Make predictions on new data

## Usage

Run the notebook and input patient health metrics to get a prediction result.
